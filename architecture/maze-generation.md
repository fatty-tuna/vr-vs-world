**Maze Generator of fantasticness!**

**MazeGenerator(size)**
Creates and generates the maze, only likes odd numbers though for size

**MazeGenerator.MazeBlocks**
Gets the maze block array

**MazeGenerator.getImage()**
Lets pretend that this works

**MazeGenerator.getHumanImage()**
Returns an image canvas the a human can understand




**MazeBlock()**
Creates a maze block

**MazeBlock.sides**
Gets the side information for the maze block

**MazeBlock.getValue()**
Returns a binary based value to replace .sides (1 - up, 2 - right, 4 - down, 8 - left)

**MazeBlock.getColour()**
Like MazeBlock.getValue(), but a colour
